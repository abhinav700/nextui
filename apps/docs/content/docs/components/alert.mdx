## API

### Alert Props

| Attribute        | Type                                                                         | Description                                                  | Default   |
| ---------------- | ---------------------------------------------------------------------------- | ------------------------------------------------------------ | --------- |
| title            | `string`                                                                     | Title for alert                                              | -         |
| description      | `ReactNode`                                                                  | Description for alert message                                | -         |
| color            | `default` \| `primary` \| `secondary` \| `success` \| `warning` \| `danger`  | The button color theme.                                      | `default` |
| radius           | `none` \| `sm` \| `md` \| `lg` \| `full`                                     | The button border radius.                                    | `md`      |
| isCloseable      | `boolean`                                                                    | Whether the close button should be displayed.                | `true`    |

### Alert Events

| Attribute    | Type                        | Description                                                 |
| ------------ | --------------------------- | ----------------------------------------------------------- |
| onClose      | `() => void`                | Handler that is called when the modal is closed.            |
